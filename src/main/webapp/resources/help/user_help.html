  1、 在线学习系统有什么功能？【返回目录】
该在线学习学习系统提供教师和学生两个角色。主要有在线课文学习、在线测试、在线作业三大功能。教师可以发布课文及其课文附件、发布测试题、发布课外作业、管理学习资料及其查看成绩的功能。学生可以在线学习课文，在线测试，在线作业及其查看相应成绩的功能。此外，该系统还拥有完善的角色注册管理个人信息的功能。
  2、 用户如何使用该系统？有什么具体流程或者操作步骤？ 【返回目录】
师和学生两类用户角色。
教师使用有如下步骤：
  • 注册账号；
  • 登录系统；
  • 添加课程信息（初次使用），二次使用时，可以不添加课程信息。若有新课程时再添加；
  • 发布学习内容；
  • 管理学习内容；
  • 学生成绩管理。
学生使用有如下步骤：
  • 注册账号；
  • 登录系统；
  • 选择老师；
  • 学习课文、在线测试、在线作业
  • 成绩查看。
  3、 用户初次使用系统时，如何获取登录帐号？【返回目录】
该在线学习系统有教师和学生两类用户。学生用户首次使用该系统可以进入学生登录界面点击【注册帐号】即可进行注册。教师用户首次使用该系统进入教师登录界面点击【注册账号】进行账号注册。
  4、 用户注册帐号时，有什么注意事项？【返回目录】
当用户进入账号注册界面时，会看到登录帐号，密码，及其其他基本信息。该系统要求用户输入所有信息才可已进行注册。系统要求帐号不能重复，因此用户在输入登录帐号后，系统会自动判断该帐号是否存在，及时提示用户。另外，系统还要求电子邮箱格式正确。若所有信息按要求输入正确即可点击【注册】按钮进行账号注册。若注册成功系统自动跳到登录界面进行登录。若注册失败，系统会给于提示帮助信息。
  5、 用户如何修改自己的信息及其密码？【返回目录】
用户进入系统后，在系统右上角可以看到自己的基本信息，学生可以点击【密码修改】进行密码修改。密码修改时，用户需输入原始密码及其两次新密码进行密码修改。若未按要求正确填写，系统会给予提示。密码修改成功后系统自动跳转到登录界面。
